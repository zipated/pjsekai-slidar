# 使用機器
- Arduino Pro Micro 5V x1
- MPR121 触摸传感器模块 x1
- WS2812B(LED灯带) x1
- 3mm乳白色亚克力板 x1
- 3mm透明亚克力板 x1
- 静電容量式センサ用透明フィルム ３枚セット（透明导电膜（应该是？））  x1
- 其他
    - 电源(LED灯带用)
    - DC插口
    - Si8640BT(数字信号隔离模块)
      - 将Arduino的控制电源和LED灯带的电源分开
    - 导线
    - 面包板
